# Mini Me — Reference Links & Inspiration

## Art Style References

### Isometric Pixel Rooms (Primary Inspiration)
- **Google Images search**: "pixel rooms" — isometric pixel art room dioramas, cozy lo-fi style
  - https://www.google.com/search?q=pixel+rooms&udm=2
- **Pinterest: Isometric pixel art room** — the specific reference image that locked in the isometric decision
  - https://i.pinimg.com/originals/23/9f/5a/239f5ae13db5953a196f87f864b2bb95.png
- These references show rooms viewed from a diagonal angle against dark backgrounds — the "lo-fi album cover" aesthetic

### Style Decision
- **Isometric (2.5D)** was chosen over flat 2D (Pokemon-style) and full 3D
- Isometric gives the cozy lo-fi warmth; top-down feels like a game, isometric feels like a vibe
- Every lo-fi YouTube stream thumbnail is an isometric pixel room — that's the audience overlap

### Pixel Art Asset Resources (for production)
- **itch.io**: Top-down and isometric interior asset packs — browse for reference and purchasable assets
- **TinyHouse by pixel_Salvaje**: Interactive isometric room builder (browser-based demo)
- **Penzilla's Retro Interior Pack**: 16x16 grid rooms (kitchen, bedroom, living room)
- **Envato Tuts+**: Isometric pixel art tutorials showing how rooms are built step-by-step

## Key Insight from References
The Pinterest isometric room image was the moment the art direction was locked in. That specific cozy, warm, detailed pixel room diorama IS the aspirational end state users work toward. Every furniture item, every room upgrade should feel like it belongs in that image.
